# Warriors Theme Tools (WordPress)

This plugin adds:
- Warriors brand styling across header/buttons/background
- Automatic menu-link normalization for login/signup/HQ labels
- HQ CTA shortcode
- Social links shortcode

## Install
1. Upload folder `warriors-theme-tools` into `wp-content/plugins/`
2. Activate plugin in WordPress Admin -> Plugins
3. Configure in Settings -> Warriors Theme Tools

## Shortcodes
- `[warriors_hq_cta]`
- `[warriors_social_links]`

## Recommended values
- HQ Base URL: `https://hq.pghwarriorhockey.us`
- Instagram URL: `https://instagram.com/pittsburghwarriorshockey`
- Facebook URL: `https://www.facebook.com/pittsburghwarriors/`
